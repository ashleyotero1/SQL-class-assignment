# SQL Database Query Projects

## Description
This repository contains my SQL coursework.  
The assignments demonstrates the use of aggregate functions and join statements to retrieve and analyze data from multiple tables.

## Skills Demonstrated
- Aggregate functions: COUNT(), SUM(), AVG(), MIN(), MAX()
- INNER JOIN and LEFT JOIN operations
- Grouping and filtering results

## How to Use
1. Open SQL Server Management Studio (SSMS) or another SQL environment.  
2. Open the file.  
3. Run each query to view results of aggregate calculations and join operations.
